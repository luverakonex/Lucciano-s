# Proyecto de programacion web - JavaScript 
Inventario de Productos para una tienda la misma permite registrar porductos
controlar su stocl, precio y categoria administrada en tiempo real 
utilizando una Fake **API local** en un json-server con el fin de 
simular un backend de una app en produccion. Para las llamadas 
utilizamos **Axios** tal cual nos indicaba el pdf del ennunciado del proyecto.
